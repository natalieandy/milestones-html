Command Line

When using the command line, remember that every character counts! Yes, even the spaces!

  • How to open the terminal outside of VS Code (using a Mac)
    1. Hold the command key and spacebar to open "Spotlight Search"
    2. Begin typing "terminal" into the spotlight search––it should appear as you start typing.
    3. Select the terminal application, this should open after pressing the enter key or by clicking on the application itself.
  • How to use "cd" to navigate between directories

    • Before starting, it's helpful to know that "cd" stands for "computer directory," so when we use this command, we are essentially directing the computer to the directory we intend to make changes to or otherwise need information on. "Directories" refer to what we (assumingly non-technical people) call "folders."

    1. Begin by typing the command "cd" by itself. This is the initial starting point for the computer to begin looking for subsequent directories. Imagine one big diagram that displays hirearchies, where there is one starting point (the "cd" command, or main computer directory) that breaks into different parts to form one large procedure of steps, or in this analogy, levels.
    2. You'll most likely want to navigate files within the Desktop (the area you see, decorated with files and folders and lives behind the browsers or applications you open), so your next command is "cd Desktop" to narrow in on the directories within the Desktop.
    3. To work with a particular directory on the Desktop, type "cd Desktop insertyourfilenamehere"
    4. To exit the directory you're currently working in (to back-up, if you will), you have to start over by beginning with the initial "cd" command.
    5. Happy navigating!
  • How to use "ls" to list the contents of a directory
    1. After navigating to a particular directory (be it the initial directory ("cd"), Desktop––whichever directory you chose), type "ls" to reveal the directory's contents.
  • How to use "mkdir" to create an empty directory
    1. Begin by choosing the directory in which you'd like to create your new directory in, and orient the computer there (using the "cd" commands explained earlier).
    2. Type the command "mkdir insertnameofyournewdirectoryhere"
    3. You should now be able to view your new directory within the larger directory you made it in. For example, if all you wanted to do was create a new directory on the Desktop, that the computer was oriented in the "cd Desktop" location.
  • How to use "touch" to create an empty file
    1. Similar to how we can create new directories within directories, we can create new files too! Simply type the command "touch insertnameofyournewfilehere" and your new file will appear under the directory you created it in.
  • How to use "code" to open VS Code
    1. Navigate to the appropriate directory (the one that has files you'd like to code) using the commands discussed above.
    2. Once in the appropriate directory, type the command "code ." This will open VS Code (assuming you have it downloaded on your computer) and you can begin coding!
